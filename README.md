Исходные данные (контекст):
Платформа 8.3.25, управляемые формы. 

В конфигурации создано перечисление «СтатусыДокументов». Значения перечисления:
Создан
Внутренний
Акцептован
Заблокирован

В конфигурации есть справочник «Договоры». Состав реквизитов:
Наименование (строка, 100 знаков)
Активен (булево)
НачалоДействия (дата; состав даты: только дата)

В конфигурации существует документ «Ведомость». У данного документа имеется табличная часть «Договоры». Состав реквизитов табличной части:
Договор (ссылка на справочник «Договоры»)

В конфигурации находится регистр сведений «ИсторияСтатусовДокументов». Независимый, непериодический. Состав измерений:
ДатаЗаписи (дата; состав даты: дата и время)
Документ (составной тип данных; множество ссылок на документы, включая ссылку на «Ведомость»)
Статус (ссылка на перечисление «СтатусыДокументов»)


Задание:
Создать внешнюю обработку. На форме обработки расположить две команды: «Заполнить нужное» и «Удалить лишнее».

Функционал команды «Заполнить нужное»:
Программно создать таблицу и заполнить её (1) ведомостями (ссылками), которые подходят под следующее условие:
дата начала действия первого активного договора в табличной части ведомости равна (или не равна1) минимальной (или максимальной2) дате начала действия договора в этой табличной части
Вывести таблицу на форму обработки, показав только колонку с ведомостями (ссылки)
Упорядочить ведомости по возрастанию даты документов
Выделить (2) светло-зелёным цветом строки, в которых последний статус ведомости равен значению перечисления «Акцептован» (согласно данным регистра сведений «ИсторияСтатусовДокументов»)

Функционал команды «Удалить лишнее»:
Удалить строки с ведомостями, у которых в табличной части чётное количество договоров
Вывести информацию (3) с удалёнными номерами (сортировка - по возрастанию номера ведомости)
формат: «Удалено документов: 3 (БТ02-000256, БТ02-000512, ЛР04-000128).»
